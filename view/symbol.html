Primitive

오브젝트가 아닌 값으로 string, number, boolean, null, undefined, symbol 이 있다.

Symbol 생성

new 연산자를 사용하여 생성할 수 없다. 

let sym = Symbol();

const sym = Symbol();
console.log("1:", sym); // 1: Symbol()
console.log("2:", typeof sym); // symbol
console.log("3:", Symbol("주석")); // Symbol(주석)
console.log("4:", sym == Symbol()); // false

Symbol 값 변경

Symbol()로 생성한 symbol 값은 변경할 수 없다. 문자열을 연결할 수 있으나 String() 또는 toString()을 사용해야 한다. 

let sym = Symbol();

try {
    +sym;
} catch (e) {
    console.log("+sym 사용 불가");
};

try {
    sym | 0;
} catch (e) {
    console.log("sym | 0 사용 불가");
};

// +sym 사용 불가
// sym | 0 사용 불가

let sym = Symbol();
try {
    sym + "문자열";
} catch (e) {
    console.log("문자열 연결 불가")
};

console.log(String(sym) + "연결" );
console.log(sym.toString() + "연결");

// 문자열 연결 불가
// Symbol() 연결
// Symbol() 연결

Symbol 오브젝트 생성

Object(123)과 같이 Object() 파라미터에 123을 지정하면 Number 오브젝트를 반환한다. 마찬가지로 Object()에 Symbol 값을 지정하면 Symbol 오브젝트를 반환한다. 

let sym = Symbol("123");
const obj = Object(sym);
console.log(obj); // Symbol {[[PrimitiveValue]]: Symbol(123)}

console.log(obj == sym); // true
console.log(obj === sym); // false
